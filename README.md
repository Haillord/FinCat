# AI Категоризатор банковских выписок

Полноценное веб-приложение на Python для автоматической категоризации банковских выписок с помощью искусственного интеллекта.

## Особенности

- 📁 **Поддержка форматов**: Excel (.xlsx, .xls) и CSV (.csv)
- 🤖 **AI-обработка**: Использует OpenAI GPT-4o-mini для умной категоризации
- 🎨 **Современный интерфейс**: Tailwind CSS для красивого и адаптивного дизайна
- 🔒 **Безопасность**: API-ключ загружается из .env файла
- ⚡ **Интерактивность**: Фильтрация, поиск и красивая таблица результатов
- 🔄 **Очистка данных**: Автоматическое удаление "мусора" из описаний транзакций

## Требования

- Python 3.8+
- OpenAI API ключ (или DeepSeek API ключ)

## Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
# Клонируйте репозиторий (если используете Git)
git clone <ваш_репозиторий>
cd bank-statement-categorizer

# Или просто создайте папку и скопируйте файлы
mkdir bank-categorizer
cd bank-categorizer
```

### 2. Установка Python зависимостей

```bash
pip install -r requirements.txt
```

**Если команда pip не распознается:**

```bash
python -m pip install -r requirements.txt
```

Или:

```bash
py -m pip install -r requirements.txt
```

**Если возникают проблемы с pandas:**

```bash
python -m pip install --upgrade pip setuptools wheel
python -m pip install pandas==2.0.3
python -m pip install flask flask-cors openai python-dotenv werkzeug
```

**Для CSV файлов (без pandas):**
Приложение может работать с CSV файлами без pandas. Если pandas не установлен, приложение автоматически использует встроенную обработку CSV.

**Для Excel файлов:**
Для работы с Excel файлами (.xlsx, .xls) требуется pandas. Если pandas не установлен, приложение предложит использовать CSV формат.

**Если pandas не устанавливается:**
Приложение будет работать только с CSV файлами. Для Excel файлов используйте конвертацию в CSV или установите pandas вручную.

### 3. Настройка API ключа

Создайте файл `.env` в корне проекта:

```bash
copy .env.example .env
```

Откройте файл `.env` и вставьте ваш API ключ:

```env
OPENAI_API_KEY=sk-your-openai-api-key-here
```

**Где получить API ключ:**
- OpenAI: https://platform.openai.com/api-keys
- DeepSeek: https://platform.deepseek.com/api_keys

### 4. Запуск приложения

```bash
python app.py
```

Приложение запустится на `http://localhost:5000`

5. **Поиск**: Используйте строку поиска для быстрого нахождения нужных транзакций

## Формат файлов

Приложение автоматически определяет столбцы в вашем файле. Поддерживаются следующие названия столбцов:

**Дата:**
- `date`, `дата`, `transaction_date`, `date_transaction`

**Описание:**
- `description`, `описание`, `description_transaction`, `transaction_description`, `назначение`

**Сумма:**
- `amount`, `сумма`, `sum`, `transaction_amount`, `amount_transaction`

## Безопасность

- **API ключ**: Хранится в .env файле, не в коде
- **Файлы**: Временные файлы автоматически удаляются после обработки
- **Валидация**: Проверка форматов файлов и размеров

## Технические детали

### Архитектура

- **Backend**: Flask (Python)
- **Frontend**: HTML + JavaScript + Tailwind CSS
- **AI**: OpenAI GPT-4o-mini
- **Обработка данных**: Pandas

### API Endpoints

- `GET /` - Главная страница
- `POST /upload` - Загрузка и обработка файла
- `GET /api/categories` - Получение списка категорий

### Обработка данных

1. Загрузка файла (Excel/CSV)
2. Автоматическое определение столбцов
3. Очистка описаний от "мусора"
4. Пакетная отправка в OpenAI API
5. Парсинг и валидация результатов
6. Возврат структурированных данных

## Возможные проблемы и решения

### Ошибка: "OPENAI_API_KEY not found"
**Решение**: Убедитесь, что файл `.env` создан и содержит ваш API ключ

### Ошибка: "Could not find required columns"
**Решение**: Проверьте, что ваш файл содержит столбцы с датой, описанием и суммой. Приложение поддерживает русские и английские названия столбцов.

### Ошибка: "Error calling OpenAI API"
**Решение**: Проверьте ваш API ключ и подключение к интернету
